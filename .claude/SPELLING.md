## Common English Mistakes

| Correct | Incorrect |
|---------|-----------|
| cannot | can not (unless emphasis needed) |
| website | web site |
| frontend | front-end, front end |
| backend | back-end, back end |
| plugin | plug-in |
| login (noun) | log in |
| log in (verb) | login |
| setup (noun) | set up |

| e-commerce | ecommerce, E-commerce |
| open source | open-source (as noun) |
